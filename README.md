# 🛒 Shopzone – Full Stack E-commerce Web Application

🚀 A modern, fully functional **E-commerce Web Application** built using Django with real-world features like authentication, cart system, and multiple payment integrations.

---

## ✨ Live Features

🔐 **User Authentication**

* Register / Login / Logout system
* Secure session handling

🛍 **Shopping Experience**

* Add to Cart / Remove from Cart
* Dynamic Cart Updates
* Coupon & Discount System

💳 **Multiple Payment Options**

* Cash on Delivery (COD)
* UPI Payment (UPI ID + Apps like GPay / PhonePe / Paytm)
* Online Payment via Razorpay

📦 **Order Management**

* Order Placement System
* Order History Tracking

🎯 **User-Friendly UI**

* Clean & Responsive Design
* Real-world checkout flow

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/shopzone-ecommerce.git

# Go to project folder
cd shopzone-ecommerce

# Create virtual environment
python -m venv env

# Activate environment
env\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
```
